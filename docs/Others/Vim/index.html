Modes

There are three basic modes in Vim

  • Command mode
  • Insert mode: press i
  • Visual mode: it is where you visually select a bunch of text so that you can run a command/operation only on that part of the text.

Navigation

  • left
  • right
  • up
  • down
  • move the cursor left: h
  • move the cursor right: l
  • move the cursor up: k
  • move the cursor down: j
  • jump to the start of a word: w
  • jump to the end of a word: e
  • insert (before the cursor): i
  • insert (at the beginning of the line): l
  • append (after the cursor): a
  • append (at the end of the line): A
  • make a new line under the current line: o

There are six ways to exit vim

  • :w write the file, but don't exit [equivalent to save]
  • ^ as root 
  • :wq write and quit [equivalent to save and exit]
  • :q quit [fails if there is unsaved changes]
  • :!q force quit [will exit with unsaved changes]
  • :wqa save and quit [all active tabs]

To write the file, but don't exit- as root, use :w !sudo tee %

Cut, copy and paste

  • copy a line: yy
  • copy 2 lines: 2yy
  • copy to the end of the line: y$
  • paste the clipboard contents after the cursor: p
  • paste the clipboard contents before the cursor: P
  • cut a line: dd
  • cut 2 lines: 2dd
  • cut to the end of the line: D
  • cut a character: x

Finding and replacing patterns

  • search forwards for a pattern: /pattern
  • search backwards for a pattern: ?pattern
  • repeat this search in the same direction: n
  • repeat this search in the opposite direction: N
  • search for "old" and replace it with "new": :%s/old/new/g
  • use "grep" to search for a pattern in multiple files: :%s/old/new/g
